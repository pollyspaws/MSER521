## Note on srcset Attributes

Some `srcset` attributes in the HTML still reference the original URLs. These are optional responsive image variants and can be removed or updated to local images if needed. The main `src` attributes have been updated to point to local images.

## Note on Fonts

Font files (BauPro-Regular.otf, BauPro-Medium.otf, BauPro-Bold.otf) are still referenced from the original site. These can be downloaded separately if you want to host them locally, but are not included in this images folder.
